# 💎 SK CPA Command Center

Advanced CPA Affiliate Tracking System with Telegram Bot Notifications, Revenue Monitoring, Live Dashboard, GEO Reports, and Daily Auto Reports.

---

## 🚀 Features

### 🔥 Real-Time CPA Lead Tracking
- Instant lead notifications via Telegram
- Approved-only lead alerts
- Multi-network postback support

### 💰 Revenue Monitoring
- Daily Revenue
- Weekly Revenue
- Monthly Revenue
- Yearly Revenue
- Lifetime Revenue

### 📊 Reports System
- Today
- Yesterday
- This Week
- This Month
- This Year
- Custom Reports

### 🌍 GEO Wise Report
- Country-wise lead tracking
- GEO revenue monitoring
- Top GEO detection

### 🌐 Auto Network Detection
- Dynamic network detection from database
- All Networks reporting

### ⚙ Admin Settings Panel
- Daily report timing
- Approved mode control
- Alert ON/OFF
- Data refresh system

### 📈 Full Luxury Dashboard
- Revenue Cards
- Revenue Trend Graph
- Network Revenue Chart
- GEO Pie Chart
- Lead Status Chart
- Recent Leads Table
- Mobile Responsive UI

### 🕛 Daily Auto Report
- Auto report every night at 12 AM
